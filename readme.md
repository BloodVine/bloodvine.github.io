# Complex Full-Stack Application

A sophisticated web application with real-time features, encryption, and modern UI built with Python Flask backend and vanilla HTML/CSS/JavaScript frontend.

## Features

- 🔐 JWT Authentication with Flask
- 📧 EmailJS Integration for Contact Forms
- 🔥 Firebase Authentication & Firestore
- 💬 Real-time Chat with WebSockets
- 🗄️ SQLite Database with Encryption
- 📱 Responsive Design
- 🔒 Data Encryption
